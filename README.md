# Byte Soluctions — Corporate Website

A fictional corporate landing page built to demonstrate front-end development skills. Designed with a professional navy and gold aesthetic, the site presents Byte Soluctions as a full-spectrum IT services company.

## 🔧 Tech Stack

- Pure HTML5, CSS3 & Vanilla JavaScript — no frameworks or dependencies
- Google Fonts: Playfair Display + Outfit
- Fully responsive layout (mobile & desktop)

## 📄 Sections

- **Hero** — Split layout with company headline and key stats
- **Services** — Four core offerings: IT Support, Web Development, Cloud & Infrastructure, Cybersecurity
- **About** — Company values and background
- **Portfolio** — Four featured fictional projects with descriptions
- **Contact** — Form with service selector and business info

## ✨ Features

- Scroll-triggered reveal animations
- Hover micro-interactions on all cards and rows
- Sticky navigation with scroll shadow
- Form submission feedback state
- Mobile-first responsive design

## 📌 Purpose

This project was developed as a portfolio piece to showcase UI/design and front-end skills for technical interviews. The company, services, and projects are fictional.
